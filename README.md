# vue-laravel-auth
Открываем первую консоль:
 - cd путь к проекту/api
 - composer update
 - Меняем .env файл
 - php artisan key:generate
 - php artisan jwt:secret
 - php artisan serv

Открываем вторую консоль:
 - cd путь к проекту/client
 - npm install или npm i (мне прост нравится первый вариант)
 - npm run serve
